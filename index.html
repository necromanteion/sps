Installing and using Python 3.

Which operating system are you using?

  • Windows
  • Mac

  • Download Python for Windows

    We'll be using the latest version - Python 3.5.0. Grab the installer below.

    Download Python for Windows

  • Installing Python

    Double-click on the file you just downloaded.
    Make sure to check "Add Python 3.5 to PATH" before you click "Install Now".

    Without adding Python to your PATH you won't be able to install third-party code.
    Don't worry! If you forget this step you can always overwrite your installation.

  • Test it Out

    Open up a command prompt window and type "python".

    Command Prompt

    Microsoft Windows [Version 6.2.9200]

    (c) 2012 Microsoft Corporation. All rights reserved.


    C:\\> python

    Python 3.5.0 (default, Sept 13 2015, 10:11:12) [MSC v.1500 32 bit (Intel)] on win32

    Type "help", "copyright", "credits" or "license" for more information.

    >>>

    If you get something that looks like this, you're good to go! If not, you may have forgotten to check "Add Python 3.5 to PATH" during installation. If all else fails, send me an email and I'll do my best to help you out.
