  • Downloading Python

    Head over to Python.org and download the latest release - Python 3.5.0.

    If you plan to use Python's scientific libraries (e.g., Numpy, scipy), consider installing the Anaconda distribution, which has them and much more pre-packaged for your convenience. As always, you'll want the latest version - Python 3.4.
